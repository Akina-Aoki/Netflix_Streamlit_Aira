# Netflix Streamlit Project

## 📦 Project Structure

```
NETFLIX_STREAMLIT/
│
├── src/
│   └── netflix/
│       └── __init__.py
│
├── pyproject.toml
├── README.md
├── .python-version
├── .gitignore
```

---

## 🚀 Setup

### 1. Initialize project

```bash
uv init --package netflix --python 3.13
```

---

### 2. Move package into `src` layout

```bash
mkdir src
mv netflix src/
```
